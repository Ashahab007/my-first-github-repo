What is Box Model?

Box Model contains following properties

  • Margin = Outer Space
  • Border = Boundary
  • Padding = Inner Space
  • Heigt x Width = Content Size

Properties of flexbox

  • flex-direction: column, row-reverse, column-reverse
  • flex-wrap: wrap
